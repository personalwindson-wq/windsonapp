# Windson Personal Trainer PWA

## Project Summary
PWA para gestão de personal trainer — clientes, agendamentos, treinos, avaliações físicas e financeiro. Hospedado no Netlify, backend no Supabase (auth OAuth Google/Apple + RLS). Interface mobile-first com tema dark gold.

## Tech Stack
- **Frontend:** HTML/CSS/JS vanilla, TailwindCSS, ES Modules
- **Backend:** Supabase (PostgreSQL + Auth + RLS)
- **Hosting:** Netlify (CDN)
- **Fonts:** Google Fonts (Inter, Sora, Space Grotesk, Material Symbols)
- **PWA:** Service Worker, Web App Manifest

## Current State
App funcional em produção. Performance Lighthouse mobile: ~52. Foco atual: melhorar design visual e performance de carregamento.
